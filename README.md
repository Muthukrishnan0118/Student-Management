# Student Management Backend

A simple REST API for managing student data using Node.js, Express, and MongoDB.

## Features
- Add Student
- View All Students
- View Student by ID
- Update Student
- Delete Student

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
